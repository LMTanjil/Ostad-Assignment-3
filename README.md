# Ostad-Assignment-3

This is the repository for the Ostad learning platform's third module assignment, focusing on JavaScript fundamentals.

---

## 📋 Topics Covered
* **Variables & Objects:** Storing complex data and accessing object properties.
* **Data Types:** Utilizing the `typeof` operator to check primitive and structural types.
* **Arithmetic Operations:** Creating functional equations for fundamental calculation workflows.
* **Conditionals:** Implementing functional dynamic grading metrics via `if / else if / else` structures.
* **Loops:** Iterating sequentially using both `for` and `while` loops.

---
